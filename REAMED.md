# Galerie d'Inspiration Créative

Projet M2 CNM / UDBL — Application de recherche d'images (Unsplash API)

## Fonctionnalités
- Recherche d'images via Unsplash API (fetch + async/await)
- Affichage type Pinterest (masonry), responsive
- Lightbox/modal pour voir l'image en grand
- Système de favoris persistants via LocalStorage
- Gestion des états de chargement et erreurs
